JDEEでJavaDoc参照の設定

  • Java
  • JDEE
    • 2010年06月09日
    ツイート

    jdkのディレクトリにJavaDocが入っていなかったので、sunのサイトからダウンロード。

    http://java.sun.com/javase/downloads/index.jsp

    ここの「Java SE 6 Documentation」からダウンロード。ダウンロードの際、言語はJapaneseを選択するように。

    ダウンロードしたアーカイブを解凍するとjaというフォルダができるので、それをjdkのディレクトリに移動し、フォルダ名をdocsにリネーム。

    ここではjdkディレクトリは/usr/lib/jvm/java-6-sunなので、以下のコマンドを実行した。

    sudo mv ~/Download/ja /usr/lib/jvm/java-6-sun
    cd /usr/lib/jvm/java-6-sun
    sudo mv ja docs

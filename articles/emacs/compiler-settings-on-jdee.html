    環境変数JAVA_HOMEにjdkのディレクトリを、JAVA_VERSIONにjdkのバージョンを入れておく。

    今回はsunのjdkを設定したいので、予めインストールしておく。

    .zshrcに以下の内容を記述（他のシェルを使用しているなら、そのシェル用の設定ファイルに。）。

    export JAVAHOME=”jdkディレクトリ”
    export JAVA    VERSION=”バージョン”

    今回は以下の様に設定した。

    export JAVAHOME=”/usr/lib/jvm/java-6-sun”
    export JAVA    VERSION=”1.6.0.20″

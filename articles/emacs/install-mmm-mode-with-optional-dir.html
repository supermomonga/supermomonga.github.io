mmm-modeを指定ディレクトリにインストールするメモ

ツイート

基本的には emacs で PHP-mode と mmm-mode を使う こちらの通りに進めればいいけど、elispは~/.emacs.d/elisp/に集約させたいので、./configure実行時にオプションを指定してやった。

./configure -with-lispdir=~/.emacs.d/elisp/mmm-mode/ ちなみに ./configure –help で各種オプションの説明を見られるので他のelispをインストールする際とかも使えると思う。 よし。

Windows 7でcaps lockキーのキーマップをCtrlキーに変更する方法

  • Windows
    • ツイート

    Windowsメニュー ＞ ファイル名を指定して実行.. ＞ 「regedit」でEnter そうするとレジストリエディタが開く。 「HKEY_LOCAL_MACHINESSYSTEMCurrentControlSetControlKeyboard Layout」というフォルダを開く。 右側のエリアを右クリックし、「新規」＞「バイナリ値」として、新しい項目を作成。 「新しい値 #1」という項目ができるので、名前「Scancode Map」に変更。 バイナリ値を上記画像の様に修正。 再起動すれば変更が反映されているはず。
